# Windows Host Inventory Reporter

A read-only PowerShell toolkit for Windows host inventory reporting.

## Features

- OS and build summary
- Computer model and memory summary
- Disk summary
- Network adapter summary
- CSV, JSON, and HTML output

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Host_Inventory_Reporter.ps1
```

## Safety

Diagnostic-only. It reports local system context for support review.
